N3 语法 EPUB：A + C 的 3 种结构版本

这次不是看“页面文风”，而是看整本书怎么编排。你的目标已经很明确：既要 A 的生动记忆感，又要 C 的强对比辨析，让一个语法不是孤零零地记，而是连着相近、相反、同组的一起记住。
你现在在选的是：“整本书的骨架”——单条即对照、主条目 + 章末辨析、还是按语法家族成组记。三种我都做成可看版本了。
Version 1
单条里直接打对照
每条语法本身就是一节小课：先讲它的感觉和场景，再立刻把容易混的、相反的、强弱不同的表达并排放进去。读一页就能记一簇。

读起来像什么

ばいい 这页里，马上就会见到 たらいい / ほうがいい / べきだ。

你不会只记一个点，而是会在脑子里形成“建议系梯度”。

  • 最符合你说的“强烈对比才好记”
  • 每页都很充实，学习感最强
  • 很像“汉字 EPUB 的语法版升级”
打开 Version 1
Version 2
主条目 + 章末辨析页
每条语法页先生动地讲明白，尽量不打断阅读节奏；然后在一章末尾，再把这一组语法集中拉出来做“辨析复盘”。

读起来像什么

平时学的时候顺，复习时又有一张“总对照表”。

像一本主教材 + 每章后的小型总结页。

  • 主条目阅读体验最好
  • 适合先学后复盘
  • 比较像成熟教材的组织方式
打开 Version 2
Version 3
按语法家族成组记
不是按“一个语法一页”去想，而是按“建议系 / 决定系 / 缓和否定系”这些语法家族来编。对比最强，记忆网络感最明显。

读起来像什么

你看到的不是单个语法，而是一个“语法家族”的地图。

更像认一片星座，而不是认一颗星。

  • 对比最猛烈，最容易成组记忆
  • 很适合你说的“相同/相反一起记”
  • 辨析感最强
打开 Version 3
