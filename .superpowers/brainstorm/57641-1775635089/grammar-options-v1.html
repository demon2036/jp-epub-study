N3 语法 EPUB 原型对比

这 3 个版本都用同一批语法样本来演示：ばいい、ことになっている、わけではない。你比较的是“讲法、节奏、记忆感”，不是被不同内容带偏。
ばいい：给建议 / 提出可行做法 ことになっている：规则 / 约定 / 外部决定 わけではない：部分否定 / 不是说……
Option A
生动记忆型
最接近《教育汉字详解》的味道。先给语法“感觉”和生活画面，再补接续和限制；强调记忆抓手、场景线、易混提醒和一句话口令。

示例气质

ばいい 不是硬邦邦的“应该”，更像“这事这样做就行”。

ことになっている 像“墙上的规定 / 大家的默认安排”，不是我临时起意。

わけではない 像往回收一点：不是全盘否定，只是否定你刚刚那个过头的理解。

  • 最容易记住，最有“画面感”
  • 更适合从不会到会
  • 后续可扩成整本“语法小课书”

Option B
平衡学习型
在“好懂”和“完整”之间取中间值。每条都有核心语气、接续、限制、例句、对比，但不会做得像词典那样干。

示例气质

先一句话讲清它“听起来像什么”，再给接续骨架、语气说明、2~3 个例句和一个易混对照。

读起来像一页页精炼讲义。

  • 学习和查阅都兼顾
  • 后续扩写成本适中
  • 更适合做长期主版本

Option C
冲刺辨析型
偏考前复习。重点不是“慢慢讲明白”，而是把高频误区和近义区别摆在一起，帮助快速扫一轮、做题时别踩坑。

示例气质

会把 ばいい 跟 たらいい、わけではない 跟绝对否定直接拉出来对照。

信息密度高，但没那么温柔。

  • 复习效率高
  • 非常适合考前过一遍
  • 容易做成辨析清单
