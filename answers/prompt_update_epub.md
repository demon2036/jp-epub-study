# Prompt 更新与新 EPUB 结果

## 我的看法
将字源内容并入释义并强调场景与可记忆性是合理的，这样读者能在“意义—场景—读音”之间建立更稳定的联想链。

## Prompt 改动要点
- 释义(summary)要求补充每种读音的典型使用场景，并尽可能结合古日语来源/词源/词根说明意义来源。
- 强调输出要简洁、可铭记、可复述。
- 删除了 etymology 字段，渲染端不再输出“字源”章节。

## 重新生成结果
- 已删除 `data/kanji_db_v2.json` 与 `data/教育汉字详解.epub`，并重新生成 5 个条目：一、七、三、上、下。
- 新 EPUB 路径：`data/教育汉字详解.epub`

## 示例（节选）
- `一` 的 summary：
  “一”是“一条线”的象形字，源自古汉语数词，核心义为“单一、起点”。音读イチ多用于汉字词、序数与数量单位（如一番、一生）。音读イツ常在复合词中发生促音化，形成いっ-（如一旦、一致）。训读ひと/ひとつ用于单独数数、指某一个对象或数量。
